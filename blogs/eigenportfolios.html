Trying to speed up eigenportfolio computations with the Power Iteration

2022/11/30 | Github Repository

Eigenportfolios are a particularly salient application of linear algebra to finance: A covariance matrix of stock price returns is decomposed, and each eigenvector in the resulting orthogonal eigenbasis can be viewed as an "eigenportfolio", or a vector of weights to be assigned to stocks.

So this requires us to eigendecompose a covariance matrix, and then search for the absolute largest eigenvalue, and take its associated eigenvector as the eigenportfolio. Numerically, this is usually done via some form of QR Iteration, where Householder reflections must be computed in $ O(m^3) $ time complexity.

However, if we are only looking for the absolute largest eigenpair, then we can instead apply Power Iteration to the covariance matrix, allowing us to compute an eigenportfolio faster. In this project, I compare the speed advantage of this approach against applying np.linalg.eigh() on the covariance matrix.
